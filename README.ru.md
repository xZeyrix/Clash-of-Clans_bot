## ✨ Возможности

### 🎮 Интеграция с Clash of Clans
- **Информация о клане** — Получение детальной статистики о любом клане по тегу
- **Мониторинг войн** — Отслеживание войн в реальном времени с автоматическими уведомлениями
- **Статус войны** — Текущая информация о войне: звёзды, процент разрушения, оставшееся время
- **Автоматические уведомления** — Оповещения на ключевых этапах войны (осталось 12ч, 6ч, 3ч, 1ч)
- **Система "смертников"** — Отслеживание игроков, не атаковавших на войне

### 💬 Модерация чата
- **Защита от спама** — Настраиваемое ограничение частоты сообщений (по умолчанию: 10 сообщений за 60 секунд)
- **Фильтр нецензурной лексики** — Автоматическое обнаружение и удаление неуместных выражений
- **Система банов** — Временная блокировка с возможностью разбана администраторами
- **Белый список пользователей** — Контроль доступа для администраторов и бета-тестеров

### 🎤 Обработка голосовых сообщений
- **Распознавание речи** — Конвертация голосовых сообщений в текст с помощью Groq AI (модель Whisper)
- **Модерация контента** — Автоматическое обнаружение нецензурной лексики в голосовых сообщениях
- **Ограничение длительности** — Поддержка сообщений до 10 минут
- **Оценка времени обработки** — Прогресс конвертации в реальном времени

### 👑 Инструменты администрирования
- **Пауза/Возобновление бота** — Временное отключение ответов бота
- **Трансляция сообщений** — Отправка объявлений в клановый чат
- **Управление мониторингом** — Запуск, остановка и проверка статуса мониторинга войн
- **Управление смертниками** — Добавление, удаление, очистка и просмотр списка нарушителей
- **Управление пользователями** — Бан, кик и разбан пользователей с помощью кнопок

### 📚 Система правил клана
- **Интерактивный браузер правил** — Навигация по правилам клана с помощью встроенных кнопок
- **Категоризация правил** — Основные правила, ЛВК, Клановые войны, События, Рейды, Роли, Наказания
- **Многостраничность** — Удобная навигация между разделами правил

## 📋 Требования

- **Python 3.10+**
- **Токен Telegram бота** (от [@BotFather](https://t.me/BotFather))
- **Учётные данные Clash of Clans API**:
  - Email/Пароль для режима разработки, **ИЛИ**
  - API токен для продакшена ([developer.clashofclans.com](https://developer.clashofclans.com/))
- **Ключ Groq API** (для распознавания речи) — Получите на [console.groq.com](https://console.groq.com/)

```bash
python-dotenv>=1.2.1
```

## 🚀 Установка

### 1. Клонирование репозитория

```bash
git clone <адрес-репозитория>
cd Clash-of-Clans_bot
```

### 2. Создание виртуального окружения

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# Linux/MacOS
python3 -m venv venv
source venv/bin/activate
```

### 3. Установка зависимостей

```bash
pip install aiogram coc.py groq python-dotenv
```

### 4. Создание файла окружения

Создайте файл `.env` в корне проекта:

```env
# Настройки Telegram
TELEGRAM_BOT_TOKEN=ваш_токен_бота
NOTIFICATION_CHAT_ID=id_основного_чата
COMMUNICATION_CHAT_ID=id_чата_для_общения
DEV_NOTIFICATION_CHAT_ID=id_чата_разработчика

# Настройки администраторов
ADMIN_USER_IDS=123456789,987654321
DEVELOPER_USER_ID=123456789

# Clash of Clans API
COC_EMAIL=ваш_email@example.com
COC_PASSWORD=ваш_пароль
COC_API_TOKEN=токен_для_продакшена
CLAN_TAG=#ТЕГ_ВАШЕГО_КЛАНА

# Groq API (для распознавания голоса)
GROQ_API_KEY=ваш_ключ_groq_api
```

## ⚙️ Настройка

### Редактирование `config.py`

```python
# Режим разработки
DEV_MODE = True  # Установите False для продакшена

# ID администраторов
ADMIN_IDS = [123456789]  # Ваши Telegram ID пользователей

# Настройки модерации
# Анти-спам: 10 сообщений за 60 секунд (настраивается в main.py)
# Длительность бана: 3600 секунд (1 час)
```

### Настройка правил клана

Отредактируйте тексты правил в `data/texts.py`:
- `RULES_MAIN` — Основные правила клана
- `RULES_CWL` — Правила Лиги Клановых Войн
- `RULES_CW` — Правила обычных Клановых Войн
- `RULES_EVENTS` — Правила событий
- `RULES_RAIDS` — Правила Рейдов Столицы
- `RULES_ROLES` — Описание ролей
- `RULES_PENALTIES` — Система наказаний

### Настройка банвордов

Отредактируйте фильтры нецензурной лексики в `data/texts.py`:
- `BAN_WORDS` — Короткие запрещённые слова (точное совпадение)
- `BAN_LONG` — Длинные запрещённые фразы (поиск подстроки без пробелов)
- `BAN_LIGHT` — Лёгкие предупреждения (без мгновенного бана)
- `BAN_TRIGGERS` — Триггерные слова для модерации

## 🎯 Использование

### Запуск бота

```bash
python main.py
```

### Режим разработки vs Продакшен

**Режим разработки** (`DEV_MODE = True`):
- ✅ Использует email/пароль для аутентификации CoC API
- ✅ Все сообщения идут в чат разработчика
- ✅ Только ID разработчика может использовать бота
- ✅ Полезно для тестирования без влияния на продакшен

**Режим продакшена** (`DEV_MODE = False`):
- ✅ Использует API токен для CoC API (более стабильно)
- ✅ Отдельные чаты для уведомлений и общения
- ✅ Требуется белый список пользователей (AllowedUsersMiddleware)
- ✅ Готов к развёртыванию

### Добавление разрешённых пользователей (режим продакшена)

При работе в продакшене необходимо добавить пользователей в белый список. Добавьте их ID в разрешённый список или реализуйте собственную логику в `utils/filters.py`.

## 🔧 Расширенная настройка

### Мониторинг войн

Система мониторинга войн автоматически:
1. Обнаруживает начало войны (фаза подготовки)
2. Отправляет уведомления при начале дня сражения
3. Отслеживает игроков, которые не атаковали
4. Отправляет напоминания за 12ч, 6ч, 3ч, 1ч до конца
5. Сообщает финальные результаты после окончания войны

**Настройка** (в `services/coc/monitor.py`):
- Интервал проверки: 30 секунд
- Пороги уведомлений: 12ч, 6ч, 3ч, 1ч

### Система модерации

**Анти-спам** (в `main.py`):
```python
antispam = AntiSpamMiddleware(
    moderation, 
    rate_limit=10,      # Максимум сообщений
    time_window=60      # За 60 секунд
)
```

**Длительность бана** (в `main.py`):
```python
moderation = ModerationSystem(ban_time=3600)  # 1 час в секундах
```
